# DDAI Network Guide

## Overview
DDAI Network operates as an AI-Powered DePIN validator node ecosystem. By running their extension, you help validate transactions and provide necessary compute power to various AI services.

## Why Run DDAI?
- **Validator Role:** Unlike simple scrapers, DDAI focuses heavily on validation, which often commands higher reward multipliers.
- **AI Ecosystem:** The bandwidth and compute power are specifically tailored for AI model processing.
- **Background Operation:** Once set up, it requires zero daily maintenance.

## How to Get Started
1. **Register:** Create an account on the [DDAI Network Dashboard](https://ddai.network/).
2. **Install the Node:** Download the DDAI Chrome extension.
3. **Authenticate:** Pin the extension to your browser taskbar and log in.
4. **Monitor Yield:** Your node status should reflect as "Active" or "Connected" inside the extension popup, and points will accrue over time.
