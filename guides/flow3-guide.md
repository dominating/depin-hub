# Flow3 Network Guide

## Overview
Flow3 Network is a decentralized wireless and AI-focused monetization platform. It allows users to earn daily passive income by contributing minimal compute and network bandwidth through a lightweight extension.

## Why Run Flow3?
- **Daily Activities:** Engage with simple daily quests or activities on their dashboard to boost rewards.
- **Low Overhead:** The extension is extremely lightweight, meaning it won't interfere with daily browsing or intensive gaming.
- **Early Stage:** Running early testnet or beta extensions often secures higher airdrop allocations.

## How to Get Started
1. **Sign Up:** Go to the [Flow3 Network website](https://flow3.network/).
2. **Add to Chrome:** Install the Flow3 Extension to your browser.
3. **Connect Account:** Log into the extension using your newly created account.
4. **Earn:** Check the dashboard daily for extra reward boosts and leave the node active in the background.
